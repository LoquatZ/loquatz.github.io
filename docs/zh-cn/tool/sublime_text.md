### 安装插件
1. 快捷键 Mac `Cmod+Shift+P` Win `Ctrl+Shift+P` 调出窗口
2. 输入 `Install Package Control` 回车，安装包管理模块
3. Perferences -> Package Settings -> Package Control -> Settings
4. 粘贴Json配置
``` json
{
    "bootstrapped": true,
    "in_process_packages":
    [
        "Codecs33",
    ],
    "installed_packages":
    [
        "A File Icon",
        "auto-save",
        "AutoFileName",
        "ChineseLocalizations",
        "Codecs33",
        "Color Highlighter",
        "ConvertToUTF8",
        "DashDoc",
        "Emmet",
        "Google Dictionary",
        "jQuery",
        "JsFormat",
        "Open URL",
        "Package Control",
        "SideBarEnhancements",
        "Smali",
        "Vue Syntax Highlight",
    ],
}
```
5. 退出重启
