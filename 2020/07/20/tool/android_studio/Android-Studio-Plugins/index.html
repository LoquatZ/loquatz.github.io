发表于 2020-07-20 更新于 2021-04-27 分类于 Android Studio ， Plugins 阅读次数：

🔧 Android Studio 好用的插件汇总

一、插件使用方式

1. Preference -> Plugins -> Marketplace -> 搜索插件名 -> Install

二、插件汇总

1.Gson格式化工具

🧰 快捷创建 Json 字符串快速转成 JavaBean 对象
Studio版本/语言 Java Kotlin
<= 4.0 GsonFormat JSON To Kotlin Class
>=4.1 GsonFormatPlus 同上 ↑

2.Android ButterKnife Zelezny 快捷创建实例化控件

🔆 配合 butterknife 实现注解，不用再手动实现 findViewById 了。
Studio版本/语言 Java Kotlin
<= 4.0 Android ButterKnife Zelezny 使用自带Android Kotlin 扩展
>=4.1 同上 ↑ 同上 ↑

3.Lifecycle Sorter

🥳 可以根据Activity或者fragment的生命周期对其生命周期方法位置进行先后排序，
    windows 快捷键Ctrl + alt + K . Mac 快捷键 option + command + K
Studio版本/语言 Java Kotlin
<= 4.0 Lifecycle Sorter Kotlin Code Sorter
>=4.1 无 同上 ↑

4.Git代码忽略管理.ignore

🤡 智能编辑.gitignore
无版本区别直接搜索 .ignore

5.IdeaVim

📚 在Idea中使用Vim编辑
无版本区别直接搜索 IdeaVim

6.Rainbow Brackets

🌈 彩虹屁 {} 对应不同的颜色
无版本区别直接搜索 Rainbow Brackets

7.Arouter Helper

⛺️ Arouter快捷跳转
Studio版本/语言 Java Kotlin
<= 4.0 ARouter Helper 无
>=4.1 无 无

8.Translation

🍚 非常好用的翻译工具
无版本区别直接搜索 Translation

10.Gradle Killer

🔫 快速杀死Gradle进程
Studio版本 名称
<= 4.0 Gradle Killer
>=4.1 无
